The CSS Flexbox offers a one-dimensional layout. It is helpful in allocating and aligning the space among items in a container. CSS Grid Layout, is a two-dimensional grid-based layout system with rows and columns, making it easier to design web pages without having to use floats and positioning. Like tables, grid layout allow us to align elements into columns and rows.

Tailwind CSS is a utility-first CSS framework for building custom user interfaces rapidly and efficiently.Bootstrap is an open-source component based framework that is used to design responsive websites better, faster, and easier. It is a beginner-friendly and the most popular open-source framework

The CSS box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages. It can be used as a toolkit for customizing the layout of different elements.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. Example: "header" tag
